## âš™ï¸ How the Quiz Works

1. Questions are stored as JavaScript objects.
2. Each question is rendered dynamically in the DOM.
3. When a user selects an answer:
   - The app instantly marks it as correct or incorrect.
   - The correct answer is highlighted.
   - Further input is disabled.
4. The **Next** button loads the next question.
5. At the end, the final score is displayed with a replay option.

This mirrors how real quiz systems handle state and user interaction.
